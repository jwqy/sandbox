Project Name : Dasmoto's Arts & Crafts
Project Description: 
	This project is required under Web Foundation/Developing Websites Locally syllabus.

	In this project, you’ll build a simple website for a fictional arts and crafts store using the tools on your own computer. 
	At a high-level, this project will require the following:
	1) A folder structure that makes sense for the project
	2) An HTML file
	3) A CSS file

	To successfully complete the project, you’ll require the 4 images provided
	The rest of the page’s styling (font sizes, colors, etc.) is outlined in the given design spec.
